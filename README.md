# BTC-Watcher
BTC (Bitcoin) Watcher in Python with curses.
<br><br>
<ins>Features:</ins>
- Settings for high and low prices watchlist.
- Audio sound and email sending  on reaching high or low prices.
- Customizable refresh rate
